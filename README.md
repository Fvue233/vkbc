## Overview
This compatibility mod enables two-way integration between Vinery and Kaleidoscope Tavern:
- Cross-storage for all bottles, compatible with both mods’ cabinets/racks.
- Grapes from Vinery can be pressed directly in Kaleidoscope Tavern’s fruit bowls.
- Brewing barrels are fully cross-compatible, supporting each other’s recipes.
- Adds Vinery grape and apple juice barrels (currently reusing Kaleidoscope Tavern textures).
- No core changes to either original mod — only optimized cross-mod data interaction for a smoother, unified experience.
- This mod was generated by AI.
